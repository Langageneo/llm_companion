# LLM Compagnon Personnel

## Description
LLM Compagnon Personnel est un assistant cognitif privé pour développeurs.  
Il utilise un modèle LLM (via OpenAI ou OpenRouter) pour agir comme un **conseiller technique et mentor numérique**, aidant à :

- Comprendre et analyser du code
- Structurer et organiser des projets
- Corriger et améliorer des raisonnements techniques
- Garder une mémoire volontaire des décisions et apprentissages

**Important** : Le LLM compagnon n’agit jamais à la place de l’utilisateur et ne possède aucune autonomie.  
L’utilisateur reste **l’architecte et décideur final**.
