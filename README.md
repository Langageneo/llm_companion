# llm_companion
LLM Compagnon Personnel Un assistant cognitif privé pour développeur, basé sur un LLM via API (OpenAI ou OpenRouter). Il agit comme un conseiller technique et mentor numérique, aidant à comprendre, structurer, corriger et améliorer le code et les projets, sans jamais agir de manière autonome. Son rôle est d’augmenter la réflexion
